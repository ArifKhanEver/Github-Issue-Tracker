
### 🚀 Essential Questions $ Answers

1️⃣ What is the difference between var, let, and const?

=> Var, let and const in javaScript differ mainly based on their scope and re-assignability. Var is function-scoped, hoisted and re-declarable whereas let and const are block-scoped which means you cannot access the value of let and const outside the block. Moreover, let allows reassignment but const does not.  
<br>


2️⃣ What is the spread operator (...)?
    
 => From a structural point of view, the spread operator acts as a copy-and-paste shortcut for data structures. It allows us to create a shallow clone of an existing array or object while merging in new data. It is a kind of mechanical way of pouring out the contents of a container so that each content can be handled separately rather than as a single group unit.
 <br><br>


3️⃣ What is the difference between map(), filter(), and forEach()?
    
 => The primary difference among these three methods lies in their returned output and intended goal. Foreach() is a simple loop that executes a function for each element without returning anything. while map() and filter() are specialized in producing a brand new array. If you think of a conveyor belt of raw materials in a factory, foreach() is a worker who inspects every items and writes down a note but leaves the items exactly where they are. Map() is a machine that takes every item and paints it a new color and place it on a brand new belt. Filter() is a quality control sensor which inspects which items are broken and places only the perfect ones onto separate and shorter belt for shipping.
 <br><br>


4️⃣ What is an arrow function?
    
=> The arrow function (=>) is a concise way of writing function in javaScript where there will be no need to use the 'function' keyword and in many cases the 'return' statement and curly braces. It is designed to make code more readable and it is powerful for short, one-line operations where the function's only job is to return a value immediately where the code will be clean and modern.
<br><br>


5️⃣ What are template literals?
    
=> A template literal (`) acts as a mini-statement where logic can be executed inside a string. The ${} syntax can hold any valid javaScript expression. This eliminates the need for repetitive plus signs and quotation marks, making code significantly cleaner and easier to read while building a dynamic messages.
